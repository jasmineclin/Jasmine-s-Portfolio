Athletic Centralized Dashboard

Current Capstone Project

Frontend Developer and Scrum Master | Tech Stack: Python Flask, HTML, CSS, Jira

My capstone team and I work alongside the Seshadri Lab at Lehigh University to develop a centralized web application for organizing wearable data collected from Lehigh athletes. Designed for use by coaches, the platform replaces fragmented team-specific systems with a unified, user-friendly interface. Built with HTML, CSS, Flask, and a connected database, the project emphasizes usability, efficiency, and scalability through Agile development and close collaboration with the Bioengineering lab.

APIs: Valid, WHOOP, Hawkin, Catapult, Beyond Pulse, and Ultrahuman.
